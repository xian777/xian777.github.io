Redis 安裝

Jan 19, 2015

2 mins read

Redis

在Linux環境上安裝Redis，最簡單的方式是用套件管理(apt or yum)，但通常不是最新版本 如果要使用最新版本，需要到官網下載原始碼編譯$ wget http://download.redis.io/releases/redis-2.8.19.tar.gz $ tar xzf redis-2.8.19.tar.gz $ cd redis-2.8.19 $ make 編譯如果碰到這個情況 make[1]: Entering directory '/tmp/redis-2.8.19/src'     CC adlist.o In file included from adlist.c:34:0: zmalloc.h:50:31: fatal error: jemalloc/jemalloc.h: No such file or directory  #include <jemalloc/jemalloc.h>                                ^ compilation terminated. 可以到deps下面先行編譯$ cd deps $ make hiredis jemalloc linenoise lua 或是直接執行以下的命令$ make MALLOC=libc Windows下的安裝，可以參考MSOpenTech在github上面的文件 https://github.
