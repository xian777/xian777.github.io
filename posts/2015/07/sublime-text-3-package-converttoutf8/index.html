[Sublime Text 3 Package] ConvertToUTF8

Jul 3, 2015

1 min read

Sublime Text 3
Sublime text 3

Sublime Text 3 預設不支援中文編碼，所以需要安裝ConvertToUTF8套件來解決這個問題 按下Ctrl + Shift + P叫出命令視窗，輸入Install Package，選擇Package Control: Install Package
