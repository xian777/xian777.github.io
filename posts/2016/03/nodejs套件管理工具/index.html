# npm指令

- 安裝資料夾 **node_modules**
``` shell

# 安裝套件
$ npm install -g 
$ npm install 

# 安裝並寫入設定檔
$ npm install  --save
$ npm install ** --save-dev

# 移除套件
$ npm uninstall -g 
$ npm uninstall 

# 搜尋套件
$ npm search 

# 列出已安裝的套件
$ npm list -g
$ npm list

# 更新套件
$ npm update -g
$ npm update

# 安裝package.json設定檔的套件
$ npm install -l
```
