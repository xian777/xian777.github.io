Sublime Text 3 安裝 Package Control

2015-06-27 14:52:00

來到 Package Control 官網，選擇Installation


選擇要安裝的版本，把下面的語法複制起來

打開Sublime Text，按下Ctrl + `打開控制台，將剛複制的語法貼上去，按Enter執行

接下來按下Ctrl + Shift + P，打開命令列，可以看到多了Package Control相關的指令

接下來安裝個套件，選擇Install Package，切換到可安裝的套件列表

輸入utf8，找到ConvertToUTF8，點擊後安裝

安裝完成

檔案會以UTF8方式儲存
